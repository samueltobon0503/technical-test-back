# Clean Architechture for Technical Test

Este proyecto da solución a la prueba técnica de desarrollador .Net y Angular Intermedio.

## Tecnologias

- Backend: .Net con EntityFramework Core y Swagger.
- Frontend: Angular 21 con Material y Tailwind.
- Base de datos: SQL Server Alojada en Azure.

## Instalación

- Clonar el repositorio
- Ejecutar el proyecto de WebApi nombrado como CleanArch.Api

Para ver el analisis de las decisiones técnicas pedidas en la pruebaconsulte el Analysis.md
